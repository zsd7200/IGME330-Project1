The Dragon Ball: Audio VisualiZer was a project for IGME-330, Rich Media Web App Dev I.

Features:
- Drag and drop or upload custom music
- Select from a variety of 10 fighters
- Select from six backgrounds
- Visual "beam" progress bar
- Find the other 6 Dragon Balls to summon Shenron
- Select from 5 audio filters
- Select from 5 visual effects
- Full screen available

Created by [Zack Dunham](https://github.com/zsd7200) and [Tristan Marshall](https://github.com/tam8217) using the Bootstrap framework.
